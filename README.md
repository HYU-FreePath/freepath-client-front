## Features
- 배리어프리 시설 정보 조회
  - 장애인 주차장, 장애인 화장실, 승강기, 건물 진입 경사로 등
- 장소 검색
  - 음성 인식으로도 검색 가능
- 웹 접근성 준수
  - 각각의 요소들을 스크린 리더가 읽을 수 있음
- 건물 구조도 조회
- PWA 지원
  - Android, iOS, iPadOS 등에서 웹 앱 설치 가능
